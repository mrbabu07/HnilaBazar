## Coupon Validation

### BEFORE (No Logging)

```javascript
const validateCoupon = async (req, res) => {
  try {
    const validation = await Coupon.validateCoupon(code, orderTotal, userId);

    if (!validation.valid) {
      return res.status(400).json({
        success: false,
        error: validation.error,
      });
    }
    // ... rest of code
  } catch (error) {
    console.error("Error validating coupon:", error);
    // No details about what went wrong!
  }
};
```

**Issues:**

- ‚ùå No visibility into validation process
- ‚ùå Hard to debug failures
- ‚ùå Can't see what data is being sent
- ‚ùå Can't see if coupon exists

---

### AFTER (With Detailed Logging)

```javascript
const validateCoupon = async (req, res) => {
  try {
    console.log("Validating coupon:", { code, orderTotal, userId });
    // ‚úÖ See exactly what's being validated

    const validation = await Coupon.validateCoupon(code, orderTotal, userId);
    console.log("Validation result:", validation);
    // ‚úÖ See the validation outcome

    if (!validation.valid) {
      return res.status(400).json({
        success: false,
        error: validation.error,
      });
    }
    // ... rest of code
  } catch (error) {
    console.error("Error validating coupon:", error);
  }
};
```

**Benefits:**

- ‚úÖ See all validation data
- ‚úÖ See if coupon is found
- ‚úÖ See validation results
- ‚úÖ Easy to debug issues
- ‚úÖ Know exactly what's failing
